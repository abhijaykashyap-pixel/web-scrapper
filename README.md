# Web Scraper

A simple Python-based web scraper that extracts tabular data from websites and converts it into CSV format for analysis and machine learning workflows.

---

## Features
- Scrapes data from static websites
- Parses HTML using BeautifulSoup
- Stores scraped data in tabular form
- Exports data to CSV
- Easy to understand and extend

---

## Tech Stack
- Python 3
- Requests
- BeautifulSoup (bs4)
- Pandas


---

```bash
git clone https://github.com/abhijaykashyap-pixel/web-scraper.git
